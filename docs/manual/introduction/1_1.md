# Haxeってなに？
Haxeは高水準のオープンソースプログラミング言語とコンパイラで構成されています。<br>
ECMAScript指向の構文で書かれたプログラムを複数のターゲット言語にコンパイルすることを可能にします。<br>
適切な抽象化を用いることで、複数のターゲットにコンパイルされる単一のコードベースを維持することが可能です。

Haxeは強い型付けですが、必要に応じて型付けシステムを逆らうことも可能です。<br>
型情報を活用することで、Haxe型システムは、ターゲット言語では実行時にのみ気づく誤りをコンパイル時に検出できます。<br>
さらに、型情報はコンパイラによって最適化され堅牢なコードを生成するために利用されます。

現在、異なるユースケースを可能にする9つのターゲット言語がサポートされています。

| 名称 | 出力タイプ | 主な用途 |
|:-----|:----------|:--------|
| JavaScript | ソースコード | ブラウザ、デスクトップ、モバイル、サーバー |
| Neko | バイトコード | デスクトップ、サーバー、CLI |
| HashLink | バイトコード | デスクトップ、モバイル、ゲームコンソール |
| PHP | ソースコード | サーバー |
| Python | ソースコード | デスクトップ、サーバー |
| Lua | ソースコード | デスクトップ、スクリプト |
| C++ | ソースコード | デスクトップ、モバイル、サーバー、ゲームコンソール |
| フラッシュ | バイトコード | デスクトップ、モバイル |
| Java | ソースコード | デスクトップ、モバイル、サーバー |
| JVM | バイトコード | デスクトップ、モバイル、サーバー |
| C# | ソースコード | デスクトップ、モバイル、サーバー |
