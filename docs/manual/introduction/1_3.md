# Hello World
以下のプログラムは、コンパイル・実行後に「Hello World」を出力します。
``` haxe
/**
    Multi-line comments for documentation.
**/
class Main {
    static public function main():Void {
        // Single line comment
        trace("Hello World");
    }
}
```
これは、上記のコードを`Main.hx`というファイルに保存し、次のようにHaxeコンパイラを実行します。<br>
`haxe -main Main --interp`<br>
すると、次のように出力されます。<br>
`Main.hx:3: Hello World`

ここから学べることは:

- Haxe言語は、拡張子が`.hx`のファイルに保存される。
- Haxeコンパイラは、`-main Main`や`--interp`などのパラメータを指定して呼び出すことができるコマンドラインツール。
- Haxe言語にはクラス(`Main`、大文字)があり、そのクラスには関数(`main`、小文字)が含まれる。
- Haxeクラスを含むファイル名は、クラス名と同じになる。<br>(この場合は`Main.hx`、`class Main {}`。)
